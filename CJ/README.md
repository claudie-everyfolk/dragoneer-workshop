## Facilitator Notes

- **Support Level:** Light. CJ is self-directed and already in daily AI use. Check in twice — once at the 10-min mark (confirm files loaded, prompt fired) and once at the 25-min mark (confirm output structure matches what he'd actually want).
- **Approach:** Frame this as "leveling up the Claude chat you already do" — same sparring instinct, but now with structured inputs and a reusable memo template. Don't over-teach Claude Code mechanics; he'll figure it out.
- **Data Confidence:** Synthetic but plausible. Expert call transcript references three companies that appear in the portfolio CSV; theme tag ("vertical-AI-for-legal") links transcript ↔ portfolio ↔ external signals. He should immediately see the cross-references.
- **Risks:**
  - He may want to plug in a *real* expert call instead — fine to allow, but warn that synthetic file is calibrated to the exercise. If he switches mid-run, expect rougher output.
  - May push past the exercise into "now do this for my whole portfolio." Good problem to have — point him to stretch goal #3.
- **Finish-Early Direction:** Push him to stretch goal #1 (theme-chasing risk score). It directly maps to his stated pain about avoiding narrative-chasing and gives him something he can show his co-head.
